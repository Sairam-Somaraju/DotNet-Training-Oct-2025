1. What is C#?
A).C# (pronounced “C Sharp”) is an object-oriented programming language developed by Microsoft that runs on the .NET framework.

2).What is the CLR in C#.
A). CLR (Common Language Runtime) is the execution engine for .NET programs.
    It manages memory, exception handling, and garbage collection.

3).  Main() method in C#:
A).It’s the entry point of a C# program.
   Execution starts from here.

4)  Namespace:
    ---------
A). A namespace groups related classes and avoids name conflicts.

5)  class:-
    ---------
A).Class is blue print of an Object.

6)  Object:
    -------
A).Object is the instance of class. which is used to access the properties of class.

7) Function:
   ---------
A). Function is a set of instructions. whenever you need to put all the related information inside the block is called function.

8) Call by Value:
   -------------
A).--> when you pass the variable as a value, a copy of the variable's value is sent to the method.
   -->Any changes made in method inside. do not effect the original value.
   --> Stored in Stack

9)  Call by reference:
    -----------------
A).-->when we pass the variable by reference, then method receives the same memory reference.
   -->Any changes inside the method affect the original value.
   -->Stored in Heap.

10) params:
    -------
A).-->The param is used to pass a number of arguments to method.
   -->when you don't know how many values will be passed.

11)Static Function:
   ---------------
A).-->static function can access the only static values which is outside of the function.
   -->we can call the function without object creation.

12)non- static function:
   --------------------
A).-->non static function can access the static values and non static values inside the non-static function.
   -->we can't call the non static method without object creation

13)static Constructor:
   ------------------
 A).-->static constructor will always executes first.
    --> Don't have any access specifier.
    --> After the remaining executes when you wrote first.

14 Copy Construction:
   -----------------
A).A copy constructor is a special constructor used to create a new object as a copy of an existing object.

15).Inheritance:
    -----------
        inheritance is a inherites properties of from existing class to new class. we have a five types.
 1).single level inheritance:
    ------------------------
         When one class inherits from one base class, it is called single-level inheritance.
 2).Multilevel inheritance:
    ----------------------
         When a class is derived from another derived class , it is called multi-level inheritance.
 3). Multiple inheritance:
     --------------------
         When a class tries to inherit from more than one base class, it is called multiple inheritance.
	 C# does not support multiple inheritance with classes (to avoid ambiguity).
         it supports multiple inheritance using interfaces.

16).Access Modifiers:
    ----------------
          1).Private: Members declared as private are accessible only within the same class.
		      They cannot be accessed from outside that class.
          2).Protected: Members declared as protected are accessible within the same class and in derived (child) classes.
          3).public: Members declared as public are accessible from anywhere in the program —
	            inside or outside the class, project,
17).Property :-
    --------
        Property is a member of a class that provides a flexble way to read, write the values of private.
18).Boxing:
    ------
         convert a value into object type. Ex:int num = 10;
						  object obj = num; 
19).UnBoxing:
    --------
         convert a object into value. Ex:int num = (int)obj;

20).Indexer:
    -------
    ->Indexer allows an object to be indexed like an array using square brackets[].
    -> Instead of accessing a vlue using a method.
    -> use "This" keyword to create indexer.
    -> Indexes can also be overload.

21).What is an Interface?
A).An interface defines a contract (methods without implementation).
  Classes implement it using the : symbol.

22).a) if statement
        ------------
     ➡ Executes a block of code only if the given condition is true.
    b) if...else statement
       -------------------
     ➡ Executes one block if condition is true, otherwise another block.
    c) else if ladder
       --------------
     ➡ Tests multiple conditions, one after another.

    e) switch statement
       ----------------
      ➡ Executes one case from multiple options based on the value.
27)  Sealed Class:A sealed class is a class that cannot be inherited. You cannot create a class that derives from a sealed class.
                  we can use when we  want to protect your code.

23).LOOPS:
    -----
    a) for loop
       -------
	➡ Executes a block of code a specific number of times.
   b) while loop
      ----------
	➡ Repeats code while a condition is true.
  c) do...while loop
     ---------------
	➡ Executes the code at least once, then repeats while condition is true.
  d) foreach loop
     ------------
	➡ Used to loop through collections or arrays.
  e) return
     -----
	➡ Exits from a method and optionally returns a value.

24). What is an Array?
A). An array stores multiple values of the same type in a single variable.
